# java-lotto

로또 미션 저장소

1. 구입금액을 입력받는다.
2. 당첨 번호(보너스 번호)를 입력받는다.
3. 로또 번호를 자동으로 생성한다.
4. 등수를 계산한다.
5. 통계를 저장하고 출력한다.

* model(domain)
    * Lotto
    * Lottos - 일급컬렉션
    * WinningLotto
    * Rank

* controller
    * LottoController
* util
    * RandomGenerator
    * InputValidator

* view
    * InputView
    * ResultView
